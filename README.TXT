== Description ==

Plugin/Gateway for wp-e-commence integration with the gateway Bradesco (http://www.bradesco.com.br/)


== Installation ==

1. Copy the file Bradesco-gateway.php to path /wp-contents/plugins/wp-e-commerce/wpsc-merchants
2. Access the WP Administration and configure the gateway whith you information from bradesco banking
